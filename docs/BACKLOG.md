## Objetivo deste documento

Este documento controla as tarefas de desenvolvimento do curso.

O backlog não altera o planejamento oficial do curso definido no PROMPT MESTRE.

Sua função é organizar:

* Produção das aulas.
* Criação dos exercícios.
* Documentação.
* Melhorias futuras.
* Organização do material.

# REGRAS DO BACKLOG

* Nenhuma tarefa deve quebrar o PROMPT MESTRE.
* Nenhuma nova etapa deve ser adicionada sem planejamento.
* O curso deve evoluir de forma progressiva.
* A qualidade do aprendizado é mais importante que velocidade.
* Cada etapa deve ser concluída antes de avançar.
